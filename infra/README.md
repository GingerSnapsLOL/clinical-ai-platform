## Infra

This directory is reserved for infrastructure-as-code and operational configuration.

Planned contents (not all implemented yet):

- `compose/` – Docker Compose files and overrides.
- `grafana/` – dashboards and provisioning.
- `prometheus/` – scrape configs and alerting rules.

Current repo uses the root `docker-compose.yml` for local development. As the project evolves, infra-related files should be moved or mirrored into this directory.
